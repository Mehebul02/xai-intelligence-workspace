# XAI – Intelligence Workspace

A premium AI SaaS product experience prototype built for the RacoAI Frontend Challenge using Next.js, TypeScript, Tailwind CSS, Framer Motion, GSAP, and React Three Fiber.

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- Framer Motion
- GSAP
- React Three Fiber
- Three.js

## Features

- Interactive Hero
- AI Intelligence Flow
- Dashboard Preview
- 3D & Scroll Animations
- Responsive Design
- Reusable Components

## Installation

```bash
npm install
npm run dev
```

Open: `http://localhost:3000`
